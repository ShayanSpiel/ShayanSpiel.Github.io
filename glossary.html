{% include components/section-head.html tag='08' title='Context Engineering' sub='The next frontier.' %} {% include components/prose-center.html text='The discipline of designing what goes into an LLM context window, including system prompts, memory, retrieval, and caching strategies. Context engineering is not prompt engineering. It is the architecture of what the model sees, remembers, and ignores.' %} {% include components/takeaway.html text='Read the Context Engineering cluster.' %}
{% include components/section-head.html tag='09' title='Prompt Caching' sub='The cost lever.' %} {% include components/prose-center.html text='Storing and reusing portions of the context window across requests to reduce latency and cost. Prompt caching is becoming part of context engineering. Providers charge less for cached tokens, making cache hits a direct cost lever.' %} {% include components/takeaway.html text='Try CacheCatch to find missed caching opportunities.' %}
{% include components/section-head.html tag='10' title='Cached Tokens' sub='The unit of savings.' %} {% include components/prose-center.html text='Tokens from a previous request that are reused in a new request's context, charged at a lower rate by providers. Cached tokens are the unit of savings in prompt caching. The more you cache, the less you pay.' %} {% include components/takeaway.html text='Try CacheCatch to audit your token waste.' %}
{% include components/section-head.html tag='11' title='Session-as-Content' sub='The foundational methodology.' %} {% include components/prose-center.html text='A methodology that turns technical founders' build sessions into publishable content without switching identities between building and creating. The build session itself is the source material. A pipeline extracts, drafts, and ships. The builder never stops working to post.' %} {% include components/takeaway.html text='Read the full methodology post or the Founder Distribution cluster.' %}
{% include components/section-head.html tag='12' title='Founder Distribution' sub='The system state.' %} {% include components/prose-center.html text='The practice of turning a founder's real technical work into public distribution surface area across multiple platforms. {% include components/section-head.html tag='13' title='Product-as-Distribution' sub='The compounding loop.' %} {% include components/prose-center.html text='When the product itself becomes the marketing channel, compounding audience and trust through usage. The product markets itself through being built.' %} {% include components/takeaway.html text='Read the Founder Distribution cluster.' %}

The smallest number of people who care enough to use, share, and pay for what you build.' %} {% include components/takeaway.html text='Read the Founder Distribution cluster.' %}
